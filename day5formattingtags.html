A Document On Meta Tag :-

The "meta" tag is used to define metadata about an HTML document, including character set, description, keywords, author, and viewport settings, always in head element. Used to achive Search Engine Optimization(SEO).

Attribute Values Description
name The name attribute is used to define the name of the property.
http-equiv This attribute is used to provide header information or value of the content attribute .
content This attribute is used to specify properties value.
charset This attribute is used to specify a character encoding for an HTML file.

character set:-

charset="UTF-8" : Specifies the character encoding for the HTML document

description:-

name="description" : describes the content of the page. This often appears in search engine results and it is one of the features of SEO.

keywords:-

name="keywords" :keywords one of the SEO features,lists of relevant keywors for page's content.

author:-

name="author" :Specifies the name of the author.

viewport:-

name="viewport" :To achieve responsiveness and UI part adjustable to screen device width.

http-equiv:-

http-equiv="refresh" content="30 : this attribute helps in refreshing the webpage for given no.of seconds.
