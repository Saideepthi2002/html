HTML5 Features:-

HTML stands for Hypertext Markup Language, the standard language for creating web pages and web applications.HTML5, the 5th version of HTML, introduces several new features.

1)Audio and Video tags:
It allows developers to embed a video or audio on their website,without relying on third party services. The tags <video> and <audio> are used.

2)Figure and Figcaption tags:
The < figure> tag is used to insert self-contained content such as photos,diagrams,illustration etc... in a document.To add the description or caption for the content inside the < figure> tag we use < figcaption> tag.

3)Nav tag:
The <nav> tag is used to define a set of navigation links.

4)Form Enhancement:
HTML 5 introduces new features for existing forms on HTML to create a more robust user experience. Some of the key enhancements in new html5 include new input typessuch as(email,date,time,url,number),new attributes(placeholder,autofocus,required) etc..

6)HTML Progress tag:
The <progress> tag is used to check the progress of a task during the execution.
